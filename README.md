# 🚀 Arduino Projects

Hey, I'm Rahul 👋  

This repo is basically my **Arduino playground** — a collection of projects I’ve built while learning electronics, embedded systems, and a bit of hardware hacking.

I like building stuff that actually works in real life — not just theory.

---

## 🧠 Why I Made This

I didn’t want to just learn Arduino from tutorials.  
So I started building projects, breaking things, fixing them, and documenting everything here.

This repo shows my **hands-on approach** to learning:
- Try → Fail → Debug → Improve → Repeat

---

## 🔧 What’s Inside

You’ll find different types of projects here like:

- 🔍 Sensor-based systems (ultrasonic, IR, etc.)
- 📡 Bluetooth-controlled devices (HC-05)
- 🤖 Small robotics experiments
- 📟 LCD / display integrations
- ⚙️ Motor + actuator control projects

Each project usually has:
Project_Name/
│── code.ino
│── circuit (if available)
│── notes (sometimes 😅)

---

## ⚡ What I’ve Learned So Far

- How to connect hardware with software (real struggle at first 💀)
- Debugging when NOTHING works
- Serial monitor is your best friend
- Power management matters more than you think
- Small wiring mistakes = big headaches
